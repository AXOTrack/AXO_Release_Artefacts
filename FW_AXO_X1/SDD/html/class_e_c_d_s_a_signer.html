Detailed Description

ECDSASigner class to sign SHA checksum implemented according to C++ singleton design pattern.

The ECDSASigner class provides functions to store a private key, to sign a SHA256 hash checksum with that key and to print out the generated signature. The class is used to sign the hash checksum calculated over the raw or displacement data to be transfered via HTTP. The calculated signature will be attached and is sended as the last HTTP frame. Because the length of the signature can vary, the length is sended as first byte of the frame, followed by the signature. If the resulting length is less than 73 byte, the rest of the frame is filled with zeros.
