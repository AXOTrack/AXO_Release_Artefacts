Detailed Description

Triggered state class derived from base class FwState.

Detailed Description
The triggered state class overrides the handler function of OtaStart and timer expired events. In this state firmware update has been triggered by OTA start event and waits for the JSON firmware info.

FwFsm::StateId handle_FwInfo ( PoolElem **  pElem)
virtual

Triggered state handler function for FwInfo events.

Parameters
pElemPointer to pointer to message of type PoolElem
Returns
Returns the state ID of the firmware download state

The function notify platform that command is accepted by calling function otaFwConfirm of the FwUpdCtrl class instance. Pointer variable pd is assigned to FwUpdCtrl class instance. Afterwards the message will be forwarded to firmware update task. Function start the download timeout timer by calling function startTimeout of the FwUpdCtrl class instance. Finally function returns the state ID eDownload.

Reimplemented from FwState.
