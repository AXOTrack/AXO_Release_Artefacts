Battery manager class derived from task base class.

Detailed Description
The Battery manager task received report requests from power manager, which trigger an ADC request from battery manager to AD converter task. The measured values will than received from AD converter task, which send them to data manager task. There they will be added to metrics JSON string. See also Message Sequence Chart Cyclic Metrics.
The AD converter class provides methods specify the ADC channels used for battery management, to retrieve the specified ADC channels and to request an report with measured values from AD converter task. Message Interaction:
  • Receive
    • Power manager task
  • Transmit
    • AD converter task
    • Data manager task

◆ task()

void task ( void  )
overridevirtual

Battery manager task function.

The function enters directly the endless loop and wait for messages. Task wakes up when a message is in the mailbox and read the message by calling mailbox read method which returns the pointer pElem to the message. All expected messages will be handled, in case of an unexpected message an error debug message will be triggered. Finally the message will be released by calling class PoolElem method free, if the pointer pElem is not explicitly set to 0.

Message Handling:

  • eTimerExpired: Unused, only debug message at debug level will be printed.
  • eAdcData: Received sensor data from AD converter task. Will forward as it is to data manager task mailbox.
  • eBattReport: Received report request frm power manager task. Battery manager request than an ADC report for the related ADC channels from AD converter task by calling member function reqAdcReport.
