Detailed Description
AXO software use Elliptic Curve Digital Signature Algorithm (ECDSA). for signing the calculated hash checksum for the data transfer of raw and displacement data via HTTP. On server side these signature and hash checksum will be verified to ensure:
  • a) the correctness of the received data
  • b) the validity of the sender identity
    The software use the mbedtls crypto library. For a detailed documentation see Mbed TLS tutorial.
