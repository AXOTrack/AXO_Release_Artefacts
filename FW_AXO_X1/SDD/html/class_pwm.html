PWM driver class implemented according to C++ singleton design pattern.

Detailed Description
The PWM driver class handles the the generation of pulse width modulated signals. It uses the public zephyr API for the PWM driver. See details in API documentation and for details of the PWM configuration in the devicetree
The PWM driver class provides an init method, methods to set PWM period and pulse width and methods to update PWM HW from settings, to start and to stop PWM generation.

◆ init()

void init ( )

The init function.

In case initialization still has been performed, indicated by flag init_done the function returns immediately, otherwise the PWM controller device structure will be retrieved by calling function device_get_binding. Than the function checks if the device is ready for usage with call of function device_is_ready. The return value of this function will be assigned to flag device_good. In case of an error a debug message at error level will be printed. Finally init_done flag is set to true.
