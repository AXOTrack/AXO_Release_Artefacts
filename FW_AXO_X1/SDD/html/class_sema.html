Detailed Description
The MQueue class implements a C++ wrapper template of the kernel message queue API provided by the Zephyr operating system. See details in Zephyr message queue documentation or in API documentation. The MQueue class provides methods to put objects of concretized type T in the queue and to get objects from the queue. Furthermore it provides a method to count the number of objects in the queue and to print statistics.
Detailed Description
The sema class implements a C++ wrapper of the semaphore API provided by the Zephyr operating system. See details in Zephyr
semaphore documentation or in API documentation. The sema class provides methods to wait for the availibility of the semaphore and a method to signal the availibility.
