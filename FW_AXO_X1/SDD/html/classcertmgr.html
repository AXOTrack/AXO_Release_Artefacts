Detailed Description

Certification management class implemented according to C++ singleton design pattern.

Detailed Description
Certification manager class is the central module for the broker switching feature. It provides methods to store received device keys abnd certificates in the file system and into the modem, methods to read and write the broker configuration file and getter function to retrieve current and desired broker. Furthermore it provides a metod to check if all precondition for the broker switching are full filled.
