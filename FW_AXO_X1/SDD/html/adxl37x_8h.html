Detailed Description
ADXL372 is connected to nRF9160 via SPI bus. Acceleration data will be received from driver via FIFO. If data is written from driver into FIFO callback function fifo_handler will be triggered by FIFO full interrupt. The fifo_handler retrieve the data from FIFO and inform own task via message. Data then will be send to eAdxlStore task for further processing. Adxl37x class uses functions provided by the ADXL372 driver, located at zephyr/driver/sensor/adxl372. These functions are included in this file by extern declaration.
Note
To use these functions from the driver, the driver implementation has to be patched. The function are mentioned only for internal use in the driver and are defined as static. For use in Adxl37x class static declaration has to be removed or commented out in patch.
