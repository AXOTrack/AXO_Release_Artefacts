#include <hash.h>

Detailed Description

Hash class to calculate SHA1 or SHA256 checksum implemented according to C++ singleton design pattern.

The Hash class provides methods to init, updata and finalize a SHA1 or SHA256 hash checksum. Furthermore getter functions to retrieve calculated hash checksum and their length and a method to print out the checksum as string in a terminal window.

◆ finalize()

void finalize ( void  )

Function to finalize SHA1 checksum calculation.

Initially the hash will be cleared with call of funtion memset. Afterwards the SHA1 hash checksum will be finalized by calling function mbedtls_sha1_finish and the context variable sha1_ctx will be released by call of function mbedtls_sha1_free, both provided by mbedtls crypto library.
