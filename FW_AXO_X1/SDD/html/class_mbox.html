Detailed Description

RTOS wrapper for mailbox support.

Detailed Description
The MBox class implements a C++ wrapper for mailbox support using the kernel message queue API provided by the Zephyr operating system. See details in Zephyr message queue documentation or in API documentation. The Mbox class provides methods to read (receive) objects of type PoolElem from the mailbox and to write (send) PoolElem objects to the mailbox. It provides an enumeration with all maibox ID's of the system, methods to register a mailbox by storing the pointer to the mailbox in an array, to retrieve the mailbox pointer by given ID Furthermore it provides a method to count the number of objects in the queue and to print statistics.
