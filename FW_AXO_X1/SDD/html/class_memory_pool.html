template<typename T, uint32_t num_blocks>
class MemoryPool< T, num_blocks >

RTOS wrapper for memory pool support.

Detailed Description
The sema class implements a C++ wrapper of the Memory Slab API provided by the Zephyr operating system. See details in Zephyr Memory Slab documentation or in API documentation. The sema class provides methods to allocate and free fixed size memory blocks of concretized type T. It furthermore provides a method to print out statistics.

◆ alloc()

T* alloc ( )
inline

MemoryPool allocation function.

Returns
A pointer to the memory block of template type T

This function allocates a memory slab object by calling function k_mem_slab_alloc provided by the operating system, which return the pointer to the memory block block_ptr and a pointer to the memory slab administration structure my_slab. Afterwards allocated memory is cleared by setting the conplete content to 0.
