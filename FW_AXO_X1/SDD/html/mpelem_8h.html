Detailed Description
Messages handled by Mbox objects are of type PoolElem. The PoolElem has the following properties:
  • The ID of the sending task
  • The message type
  • A pointer to an optional message content/buffer
  • A pointer to the message pool that allocated the message object For messages indicating a simple event (e.g. "LTE connected") the message type itself may be sufficient to convey this information. Some messages may have larger content, e.g. strings or structs. These are copied into the buffer member of the PoolElem.
