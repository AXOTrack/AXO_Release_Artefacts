Detailed Description
Calculation of displacement values based on the recorded acceleration data in z-direction. The implementation of this class is based on the calculation done by enDAQ VibrationData Toolbox Version 13.3 by Tom Irvine. To reprodice the calculation results of the toolbox a small python script has been developed which was used as the base of the implementation in this class. The python script is located at AXOTrack repository.
Because of memory restrictions the implementation on the AXO X1 device could not transfer the complete acceleration data in one vector to the alorithm, instead the acceleration data is requested in small pieces from ADXL store task, so that the calculation can be performed sequentially.
See also message sequence chart in section Displacement Calculation.
