Detailed Description
AXO devices uses the HTTPS application layer protocol for the transmission of raw and displacement data to the remote webserver. See details of HTTP and HTTPS in wikipedia. For the data transmission HTTP POST is used.
The data to be transferred consists of the following parts:
  • The HTTP POST header
  • The raw data header (first frame)
    • IMEI
    • timestamp
    • total number of data samples
    • Type of data (Raw or Displacement)
  • The x, y and z raw data samples (in the following frames)
  • OR the displacement data (in the following frames)
  • The SHA1 checksum (last frame)
    See also message sequence chart in section Raw Data Transmission. For SHA1 hash calculation the HTTP connector class uses the mbedtls crypto library.
