#include <matrix.h>

Detailed Description

Matrix helper class.

The matrix helper class provide the following fuctionality:

  • Getter function to retrieve the matrix inverse
  • Functions to print a matrix or a vector
  • Functions to initialize a matrix or a vector
  • Function to set a value at defined indices
  • Multiplication of matrix with vector
  • Substraction of vectors
  • Function to retrieve the matrix cofactors
  • Function to retrieve the transposed matrix
  • Function to retrieve the matrix determinant
