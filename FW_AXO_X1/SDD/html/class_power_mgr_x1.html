Detailed Description

Power Manager class for AXO X1 devices derived from Power Manager base class.

Detailed Description
The power manager X1 class provides the following timer:
  • BoostCtrlTimer: Timer to control boost on before initial metrics tx (eLteWakeTmo)
  • myTLteStartDelayTimerimer3: Timer to control inital LTE start delay (eLteStartTmo)
  • AdxlWakeupTimer: Timer to control adxl wakeup to measurements tx (eLteWakeTmo) It provides a method to request switching on and off the boost converter, methods to request RMS, environment, battery, HW and LTE reports for metrics collection and methods to report reset cause and boot counter to the data manager task. It furthermore implements the from base class required methods taskInit and message_handler, which are called in the task function provided by the base class. Message Interaction:
  • Receive
    • Timer interrupt callback
    • LTE connector task
    • MQTT connector task
    • RMS store task
    • Power manager task
  • Transmit
    • Boost converter task
    • RMS store task
    • SHTCX task
    • Battery manager task
    • HW Metrics task
    • Data manager task
    • Power manager task (self, transmission in timer callback)

◆ PowerMgrX1() [1/2]

PowerMgrX1 ( const char *  name = "??")
explicit

Power manager X1 task constructor.

Parameters
nameThe name of the class

The constructor register all provider tasks for metric updates by calling function registerMetricProvider provided by the power manager base class. The provider tasks are:

  • The shtcx task for temperature and humidity
  • The battery manager task for battery metrics
  • The HW metrics task for hardware metrics
  • The data manager task for LTE metrics Afterwards it allocates the timer for initial boost on, inital LTE start delay and adxl wakeup to measurements transmission.

◆ reqBoostConverter()

void reqBoostConverter ( bool  on = false)

Send resume or suspend message to boost converter task.

Parameters
onbool parameter to select message type eResume or eSuspend

The function send a resume or suspend message to the boost converter task by calling function sendMsg provided by class Mbox. The message type is set according to input parameter 'on' (eResume if on = true, eSuspend if on = false). This request is the trigger to switch boost converter on or off.
