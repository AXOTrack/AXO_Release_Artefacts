Detailed Description
This source file implements the update controller module for managing firmware updates in the AXO X1 device. The update controller handles the logic for handling OTA commands, firmware update confirmation/rejection, and managing the retry mechanism for firmware downloads. It utilizes the Finite State Machine (FSM) for coordinating different stages of the update process.
