Detailed Description

State base class.

Detailed Description
The state base class provides a pointer to a data object, which normally is the this pointer of the state machine owner class, here this pointer of firmware update control task. It furthermore provides default implementations of the handler functions, which may be overridden by the derived state classes. All state classes of the firmware download statemachine shall be derived from this base class.

◆ handle_OtaStart()

FwFsm::StateId handle_OtaStart ( )
virtual

State handler function of the state base class for OtaStart events.

Returns
Returns the state ID eNoChange

Default implementation for OtaStart event handling. Derived classes may overload this default implementation. The function reject the start command by calling function otaStartReject of the FwUpdCtrl class instance. Pointer variable pd is assigned to FwUpdCtrl class instance. Finally function returns the state ID eNoChange.
