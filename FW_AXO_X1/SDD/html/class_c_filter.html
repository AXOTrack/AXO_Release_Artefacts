Detailed Description

Digital filter class implemented according to C++ singleton design pattern.

Detailed Description
The CFilter class handle digital zero phase filtering of recoreded acceleration data. It is implemented to filter the data sequentially and piecewise as the data is received from FRAM. For zero phase filtering the data will be filtered forward and backwards. The class provides methods to initialize forward and backward filtering, to filter the data in first, Nth and last iteration and functions to provide the filter order nfilt and the length of edge transients nfact. It also provides methods to calculate the initial conditions for forward and backward filtering.

Function to start forward or backward filtering with first iteration.

Parameters
XThe input vector of type double
YThe output vector of type double

The function resize output vector Y with the same size as input vector X and initialize pointer variables to these vectors. It than initialize the vectors with the previous X and Y values with size 4 and initial values 0. Afterwards the function enters a loop over all input samples and calculate the filtered result values, store it in output vector and store the previous X and Y values for the calculation in the next iteration.
