Detailed Description
FRAM chip is connected to nRF9160 via SPI bus. FRAM is used to store the acceleration raw data. FRAM has 512 byte storage capacity. In one measurement cycle (15 measurements in an one second cycle with sampling rate of 4400Hz) the capacity of the FRAM is nearly exhausted (). Therefore a measurement cycle has been limited to 15 seconds.
To be able to calculate displacement data the FRAM raw data storage must be reduced to 5 seconds measurement, so that there is enough space to store intermediate and final displacement data. The memory layout is therefore separated in three blocks (see section Displacement Calculation for further details).

See MB85RS4MTY data sheet, page 5 for SPI connection.
