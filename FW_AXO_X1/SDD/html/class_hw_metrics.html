HwMetrics class derived from task base class.

Detailed Description
The HwMetrics class is responsible to collect status informations from HW modules and to provide them to data manager. It provides a report method to send HW metrics data to data manager task and a method to translate incomming status information from Adxl37x task into HW metrics structure. Message Interaction:
  • Receive
    • Power manager task
    • Adxl37x task
  • Transmit
    • Data manager task
    • Adxl37x task

◆ task()

void task ( void  )
virtual

Adxl37x task function.

The task function directly enter the endless loop and wait for messages. Task wakes up when a message is in the mailbox and read the message by calling mailbox read method which returns the pointer pElem to the message. All expected messages will be handled, in case of an unexpected message an error debug message will be triggered. Finally the message will be released by calling class PoolElem method free, if the pointer pElem is not explicitly set to 0.

Message Handling:

  • eHwReport: Request a HW status report. The message will be forwarded to Adxl37x task. Pointer to the pool element will be set to 0 to prevent the releasing of the message back to the pool.
  • eHwStatusAdxl37x: Status indication from Adxl37x task. Error counter will be reset to 0, message payload will be translated in HW metrics structure by calling function HandleHwStatusAdxl37x and this structure will be send to data manager task by call of funtion report.

◆ report()

void report ( )
private

Function to send a HW metrics message to data manager task.

The function request a pointer to the mailbox of data manager task and a message from message pool, set message type to eHwStatusAdxl37x, set Adxl37x task as sender, copy the HW metrics structure into payload of the message and send it to data manager's mailbox.
