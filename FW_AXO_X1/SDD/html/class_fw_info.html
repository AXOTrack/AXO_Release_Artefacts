#include <fwinfo.h>

Detailed Description

Firmware info helper class.

Detailed Description
The FwInfo parses the OTA update topic JSON string, extract the content and to store it in private string member variables. It provides methods to parse the JSON string, to validate the received file size and checksum and to split an URL into host and filename. Firthermore it provides getter function to retrieve the extracted strings.

◆ parse()

int parse ( )

Function to parse the OTA update topic JSON and extract the data.

Returns
0 in case of succeeds, -1 otherwise

The function parses the JSON string input by calling function cJSON_Parse provided by the cJSON API and store it in local variable root_obj of type struct cJSON. If parsing failed the function returns immediately with error code -1. Otherwise the objects url_obj, sha1_obj, fw_version_obj, size_obj of type struct cJSON will be extracted from root object root_obj by calling function cJSON_GetObjectItemCaseSensitive with the arguments "url", "sha1", "fw_version" and "size". The resulting object value string will be stored in the private member string variables url, sha1, fw_version and filesize before deleting the root object by call of function cJSON_Delete.

◆ splitUrl()

void splitUrl ( )

Function to split an URL into host and filename.

The function splits an URL into host and filename. For example: "http://gomera3.ddns.net/stg/app_update.bin" will be split into

  • host: gomera3.ddns.net
  • filename: /stg/app_update.bin The function copy the url into a local string tmp, search for protocol part in url, store it in member variable proto and delete it from string tmp. Than it extract host string and store it in member variable host and delete it also from string tmp. Finally the remaining filename part will be stored in member variable filename.
