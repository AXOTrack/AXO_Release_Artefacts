Detailed Description

Power Manager base class derived from task base class.

Detailed Description
The power manager task provides a timer to control the metrics TX interval (MetricsTxTimer). It provides methods to report reset cause and boot counter to the data manager task, to switch UART on and off, to register and notify the metrics providing tasks, to configure the metrics transmission interval and the task function. It furthermore declares a task init and a message handler function as pure virtual. These function must be implemented in all derived classes. Both are called in the task function, task init function at the beginning of the task function before entering the endless loop, the message handler function in the endless loop, directly after reading the message from the tasks mailbox.
