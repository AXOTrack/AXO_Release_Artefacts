◆ cmd_mux_sweep()

static int cmd_mux_sweep ( const struct shell *  shell,
size_t  argc,
char **  argv 
)
static

Callback function for console input command 'sweep'.

Function enable the muxer by calling MuxController member function enableLoad with parameter true and enters a loop over the number of configured channels. In the loop a channel is selected by call of MuxController member function selectChannel, the analog value at this channel is read by call of function adc_read, converted to milli volt by call of function adc_raw_to_millivolts and printed by call of function shell_print.
Finally the function enable the muxer by calling MuxController member function enableLoad with parameter false.

Note
For testing only: muxer configuration via shell command
