Detailed Description

RTOS wrapper for software timer support.

Detailed Description
The timer class implements a C++ wrapper of the timer API provided by the Zephyr operating system. See details in Zephyr
timer documentation or in API documentation. The timer class provides methods to start and stop timers.

Timer class constructor.

Parameters
cbThe expiry callback function of type k_timer_expiry_t
pUserPointer to the owner of the timer (normally this pointer of task)
durationTimer duration in seconds
periodTimer period in seconds for cyclic timer, default is 0 (not a cyclic timer)

The constructor initializes a timer, prior to its first use by calling function k_timer_init provided by the operating system, which assign the expiry callback function and returns the timer administration structure timer_handle. It set the owner of the timer by calling function k_timer_user_data_set also provided by the operating system and initializes the protected varaibles duration and period.
