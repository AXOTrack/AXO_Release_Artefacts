#include <mram.h>

Detailed Description

SPI MRAM class implemented according to C++ singleton design pattern.

Detailed Description
The SPI MRAM class handle the storage of the acceleration raw data in MRAM. It provides methods to initialize and erase MRAM, to read and write the data, to push MRAM into power saving mode or release from it, to dump the data and to read and ship the devcie ID.

◆ init()

void init ( )

The initialization function of the SPI MRAM class.

The function initialize the value for status register with 0x40 (Serial Number Write protected) and for the configuration register 1 with 0x01 (Unlock TB and BP[2:0], Write Protection for Augmented Area Data) and the private member variables spi_dev (SPI controller device) and spi_cfg (SPI controller configuration) by calling the functions getController and getConfig of the spibus instance spibus::inst(). Afterwards if wake up the MRAM by calling member function wakeup, read device id, status register and configuration register 1 by vall of the member functions getDeviceId, getStatusRegVal and getConfig1RegVal. If status register has not value 0x40, configuration register 1 and status register will be read by calling the member functions write_config1_reg and write_status_reg and status register will be read again by call of function getStatusRegVal. Finally device ID and content of status register will be checked. If davice ID is not 0xD9010501 or status register has not value 0x40, an exception will be raised with call of macro __ASSERT_NO_MSG.
