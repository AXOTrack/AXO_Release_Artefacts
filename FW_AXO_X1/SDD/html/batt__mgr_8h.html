Detailed Description
The battery manager class specify the ADC channels used for battery management, These are:
  • AIN1 -> ANALOG_VDD
  • AIN2 -> ANALOG_VBAT2
  • AIN3 -> ANALOG_VBAT1
  • AIN7 -> NRF_SAADC_VDD The battery manager is responsible to request the ADC measurement values from AD converter task and to report these values to data manger task, which will add them to the metrics JSON string. See also Message Sequence Chart Cyclic Metrics.
