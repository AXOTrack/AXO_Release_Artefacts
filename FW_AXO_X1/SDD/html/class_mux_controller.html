MUX Controller driver class.

Detailed Description
The MUX Controller driver class handle the selection of the connected analog inputs and connect the selcted channel to the drain pin, which acts as output and is connected to the analog input pin 14 / AIN1 of the NRF9160. The class provides methods to select the analog input channel, to enable or disable the analog input, to print the status and to retrieve the nuimber of attached analog input signals.

◆ init()

int init ( )

The MUX Controller init function.

Returns
Returns 0 in case of success, error code ENODEV otherwise

The function enters a loop over the array of muxer select lines and checks if the GPIO at index i is ready for usage with call of function device_is_ready. If a GPIO is not ready, the function returns immediately with error code ENODEV. Afterwards it checks if the muxer is enabled and the enable GPIO is not ready for usage. In this case the function returns with error code ENODEV, otherwise enable GPIO is set to inactive and functions returns 0.

Parameters

Returns

The function checks validity of the parameter ch. In case it is not in valid, the function returns immediately with error code EINVAL. Otherwise ADC channel is selected by call of function gpio_pin_set_dt and member variable current_channel_ is set to parameter ch.
