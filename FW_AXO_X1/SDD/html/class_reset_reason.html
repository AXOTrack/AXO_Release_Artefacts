Detailed Description

Reset Reason class implemented according to C++ singleton design pattern.

Detailed Description
The class provides methods to handle the reset reason, which is determined at startup by reading register RESETREAS. Register value and reset reason string are stored outside of this class in context of the caller of getter function.

Getter function to retrieve the reset reason.

Parameters
maskPointer to unsigned int variable to store content of register RESETREAS
reason_strReset reason as string representation

Function retrieve the reset reason from register RESETREAS by calling function nrfx_reset_reason_get and store the value in variable reason and assign it to pointer mask. It count the number of reasons by calling function popcount and clear register RESETREAS by calling function nrfx_reset_reason_clear. According to the content of variable reason a string variable will be filled, in case of more than one reset reasons string will be expanded. Finally string will be assigned to pointer reason_str. Both parameter will be passed to caller of this function.
