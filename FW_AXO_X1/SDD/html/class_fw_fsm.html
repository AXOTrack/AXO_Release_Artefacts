#include <fsm.h>

Detailed Description

Firmware update state machine class implemented according to C++ state machine design pattern.

Detailed Description
The class provides the four states Idle, Triggered, Firmware download and Ready and handler functions for the events OtaStart, FwInfo, DlComplete, DlFailed and timer expired.

◆ setState()

void setState ( FwFsm::StateId  newStateId)

Function to perform state transition to old state to new state.

Parameters
newStateIdThe ID of the desired new state

If the state has not changed, indicated by state ID eNoChange the function returns immediately. Otherwise the state instance of the current (old) state will be deleted and with the creator function in array stateTbl an instance of the new state will be allocated and assigned to the current state pointer myState. Also myStateId will be assigned to the ID of the new state. If the new state is not eIdle Adxl37x task will be informed that the update process is active.
