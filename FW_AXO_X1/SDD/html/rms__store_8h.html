Detailed Description
Acceleration measurement data provided by AXO X1 includes root mean square (RMS) values, hit values and a timestamp. The RMS storage provides a structure to held these values and reserves RAM to store for a configurable amount of time.
The RMS and hit values will be calculated by RMS task and send via message to the RMS storage task (see header file rms (rms.h)).
For more details see also message sequence charts in section Measurement.
