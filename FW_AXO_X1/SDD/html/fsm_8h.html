Detailed Description
This header file defines the Finite State Machine (FSM) for handling firmware updates in the AXO X1 device. It provides the state transitions and event handling mechanisms necessary for managing the different stages of firmware updates, including OTA commands, firmware info handling, download, and restart. The FSM is implemented using a set of state classes that encapsulate the behavior of each state.
