WakeupCnt ( )

Todo:
Boot Count constructor
Parameters
fnameFilename of the boot counter file

The constructor is declared as private according to singleton pattern. It is called in the static inst function to create the exclusiv only instance of the class. fname "/lfs/boot_count" is defined as default parameter in declaration of the constructor. The constructor open the file and the boot count will than be read from file. In case boot count is negative it will be set to 0. Afterwards boot count will be incremented by 1 and stored in file.
