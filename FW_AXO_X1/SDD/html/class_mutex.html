RTOS wrapper for mutex / binary semaphore support.

Detailed Description
The sema class implements a C++ wrapper of the mutex API provided by the Zephyr operating system. See details in Zephyr
mutex documentation or in API documentation. The Mutex class provides methods to wait for the availibility of the mutex and a method to signal the availibility.

int lock ( void  )

Mutex lock function.

This function locks the mutex by calling RTOS function k_mutex_lock. If the mutex is locked by another thread, the calling thread waits until the mutex becomes available. Mutex value is set to 0.

Returns

This function locks the mutex by calling RTOS function k_mutex_lock with timeout. If the mutex is locked by another thread, the calling thread waits until the mutex becomes available or until a timeout occurs.
