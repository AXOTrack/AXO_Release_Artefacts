FwFsm::StateId handle_OtaStart ( )
virtual

Idle state handler function for OtaStart events.

Returns
Returns the state ID of the triggered state

The function notify platform that command is accepted by calling function otaStartConfirm of the FwUpdCtrl class instance. Pointer variable pd is assigned to FwUpdCtrl class instance. Function start the trigger timeout timer by calling function startTimeout of the FwUpdCtrl class instance. Finally function returns the state ID eTriggered.
