Detailed Description
A mutex is simply a variable that is non-negative and is shared between threads. It is a signaling mechanism for low-level synchronization of threads and/or tasks. It uses the two atomic operations Wait and Signal for the process synchronization.
The binary semaphores or mutexes are quite similar to counting semaphores, but their value is restricted to 0 and 1. In this type of semaphore, the wait operation works only if semaphore = 1, and the signal operation succeeds when semaphore= 0.
