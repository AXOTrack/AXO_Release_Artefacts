Detailed Description

Hit detector helper class.

Detailed Description
The hit detector class compare measurement data in z direction against threshold and count the number of hits. It provides a method to identify a hit and store it in a private couter variable, a method to retrieve the number of hits and a method to reset the counter variable.

◆ input()

void input ( float  z)

Function checks if sample z fullfills hit detection condition.

Parameters
zAcceleration raw data sample in vertical direction

The function increments current sample counter n_cur and set the bool variables is_above and is_below, where is_above is set to true and is_below is set to false if input sample z is higher than hit detection threshold. Variable n_high marks the first value over the threshold and the start of the time window. If a sample value is higher than threshold and there is no time window active n_high will be set to current sample counter n_cur and a new time window has been opened. Within the time window of length max_samples the function checks if there is a value below threshold. In this case hit detection condition is met and hit counter hit_cnt will be incremented by 1 and time window will be closed by setting n_high to 0.
