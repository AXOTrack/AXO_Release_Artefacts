Detailed Description
The ADC supports up to eight external analog input channels. It can be operated in One-shot mode with sampling under software control, or Continuous mode with a programmable sampling rate. The analog inputs can be configured as eight single-ended inputs, four differential inputs or a combination of these. Each channel can be configured to select:
  • AIN0 to AIN7 pins
  • VDD_GPIO pin See details in nRF9160 Product Specification v2.0, page 198ff.
