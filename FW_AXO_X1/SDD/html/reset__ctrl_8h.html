Detailed Description
For security reasons and to ensure that the software never remains in an deadlock a cyclic power cycle reset (cold start) is established.
Power cycling is triggered on Pin P0.01 of nRF9160. The pin should be either high impedance input or logic high output during normal operation. It is externally pulled up by a 100k resistor.
Pulling P0.01 low for at least 7.5 seconds will trigger a reset signal by STM6519APAAUB6F that pulls low the enable pin of AP22913, for 200ms which in turn disables power to the entire consumer side of the board, including NRF9160, all sensors, flash memory etc. AP22913 has an output discharge function to ensure that the output is nearing 0V during off-time.
The supercap is not being discharged by this since it is located on the input side of the AP22913 load switch.
