Detailed Description
Zero phase filtering inmplementation based on Python and Matlab filtfilt function. The base of this c++ implementation is Implementation of Darien Pardinas. Eigen library is replaced by matrix operations. The filtfilt algorithm matches the initial conditions on the filter to minimise start and end transients.
