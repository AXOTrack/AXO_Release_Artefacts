Detailed Description

The pool element class that provides the messages to be send or received.

Detailed Description
A pool element is a messages which will be send to or received from tasks. A pool element is allways allocated from an existing message pool of type MsgPool. The private variable pool held the pointer to the originating pool. The pool element class provides methods for setting and getting the content of the message and for freeing itself, furthermore getter functions to retrieve the pointer to the message content and the buffer size and methods to clear the buffer content and to put data into the buffer.
