MQueue class constructor.

MQueue put function.

Returns
0 Message sent.
ENOMSG Returned without waiting or queue purged.
EAGAIN Waiting period timed out.
Parameters
dataPointer to message object. Type needs to be concretized by calling task

This function sends a message to message queue by calling function k_msgq_put provided by the operating system. Sending means putting the message object at the end position of the queue (FIFO).
