Detailed Description

Ready state class derived from base class FwState.

Detailed Description
The ready state class overrides the handler function of timer expired events. In this state the download has been finished successfully and a reset has to be performed.

The function print a debug message at warning level, perform a sleep of 3 seconds to wait for the final debug messages and perform a reset by calling function sys_reboot provided by Zephyr library API. The return statement will never reached.
